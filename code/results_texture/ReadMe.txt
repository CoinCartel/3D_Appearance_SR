This folder contains the PSNR and SSIM values of the super-resolved texture maps.
./TA_Summary_RGB.html contains all of the psnr values.
./SSIM_Summary_RGB.html contains all of the ssim values.
./table.html and ./table_ssim.html contains the values used in the CVPR19 paper.
./TA_PSNR.pkl and ./SSIM.pkl contains the values in pickle format. Do not delete them.
